# Product Overview

The Haunted Feedback Form is a spooky-themed web application that collects user feedback through an interactive form and displays a ghost-like, atmospheric response message. 

## Purpose

- Demonstrate Kiro's capabilities (specs, hooks, steering) in a hackathon setting
- Provide a simple, engaging user experience with a Halloween/spooky theme
- Showcase client-side web development without backend dependencies

## Key Features

- Interactive feedback form with themed inputs (name, spooky level, favorite creature, message)
- Personalized spooky message generation based on user input
- Dark atmospheric visual theme with neon green/purple color scheme
- Single-page experience with URL parameter-based data passing
- No backend or database required - fully client-side

## Target Audience

Developers and hackathon participants interested in seeing Kiro's features in action through a fun, themed demo application.
